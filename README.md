> <h2>Front-End 역할 및 주요 기능</h2>

<h3>조수현 (팀장)</h3>

1. 메인
2. 검색 결과
3. 유저 관심 글램핑장
4. 404
5. Git 관리

<h3>조원영</h3>

1. 로그인 및 회원가입
2. 글램핑장 후기
3. 유저 예약 내역
4. 유저 정보
5. FE 노션 관리

<h3>김민지</h3>

1. 글램핑장 상세
2. 객실 상세
3. 예약 및 결제
4. 유저 후기
